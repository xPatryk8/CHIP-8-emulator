### CHIP-8 Emulator
CHIP-8 emulator written in C and Raylib.

## Building

### Requriments
- C compiler
- Cmake 3.14 or higher

Raylib is automatically downloaded and built by CMake.

### Compilation
	cmake -B <path-to-build>
	cmake --build <path-to-build> --config Release

## Usage

    chip-8 <path-to-ROM

## Controls
  CHIP-8  | Keyboard
  1 2 3 C | 1 2 3 4
  4 5 6 D | Q W E R
  7 8 9 E | A S D F
  A 0 B F | Z X C V

**Esc:** exit
